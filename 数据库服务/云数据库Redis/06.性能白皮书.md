## 测试方法

### 压测条件

5台4C8G压测机，单片基准结果(此结果只对本次单个redis-benchmark命令，5台压测机执行基准测试结果平均值)

### 压测参数

```shell
redis-benchmark -c ${c} -n 5000000 -h 10.33.24.106 -p 9736 -d 128 -t set,get -r 5000000 -a test12345T
```
