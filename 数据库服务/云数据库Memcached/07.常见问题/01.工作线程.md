## 云数据库Memcached支持多少个线程？

Memcached实例的工作线程和规格相关。对应关系如下：

| 实例规格              | 支持线程数 |
| --------------------- | ---------- |
| 1G、2G                | 1个        |
| 4G、8G、16G、24G、32G | 2个        |
| 64G                   | 4个        |
| 128G                  | 8个        |
