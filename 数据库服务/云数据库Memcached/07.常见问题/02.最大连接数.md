## 云数据库Memcached支持的最大连接数是多少？

Memcached实例的最大连接数和规格相关。对应关系如下：

| 实例规格                      | 最大连接数 |
| ----------------------------- | ---------- |
| 1G、2G、4G、8G、16G、24G、32G | 10,000     |
| 64G                           | 20,000     |
| 128G                          | 40,000     |
