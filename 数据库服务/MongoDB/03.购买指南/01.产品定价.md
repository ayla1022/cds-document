#### 计费示例

> **说明**：以下示例的价格仅为示例，具体价格请以控制台为准。

##### 副本集计费示例

例如您在北京可用区购买如下配置的包年包月副本集实例使用 1 个月：

| 架构类型 | 实例规格 | 存储大小 |
| -------- | -------- | -------- |
| 副本集   | 2C4G     | 400      |

总费用 = 实例规格费用 + 存储空间费用 = ( 600 + ( 3.38 * 400 ) ) = 1952 元

##### 分片集计费示例

例如您在北京可用区购买如下配置的包年包月分片集实例使用 1 个月：

| 架构类型 | mongos 个数 | mongos 规格 | Config Server 个数 | Config Server 规格 | Shard 个数 | Shard 规格 | Shard 存储大小 |
| -------- | ----------- | ----------- | ------------------ | ------------------ | ---------- | ---------- | -------------- |
| 分片集   | 2           | 2C4G        | 3                  | 1C2G               | 2          | 2C4G       | 400            |

总费用 = mongos 规格费用 + Config Server 规格费用 + Shard 规格费用 + Shard 存储空间费用

##### mongos 规格费用

mongos 规格费用 = mongos 规格费用 * mongos 个数 = 200 * 2 = 400 元

##### Config Server 规格费用

Config Server 规格费用 = Config Server 规格费用 * Config Server 个数 = 300 * 3 = 900 元

##### Shard 规格费用

Shard 规格费用 = Shard 规格费用 * Shard 个数 = 600 * 2 = 1200 元

##### Shard 存储空间费用

Shard 存储空间费用  = Shard 存储空间费用 * 存储空间大小 * Shard 个数 = 2.24 * 400 * 2 = 1792 元
