## 实例相关

| API                 | 描述                                      |
| ------------------- | ----------------------------------------- |
| DescribeZones       | 获取云数据库MongoDB支持的站点区域         |
| DescribeSpecInfo    | 获取某个站点支持的MongoDB产品类型以及规格 |
| CreateDBInstance    | 创建MongoDB云数据库实例                   |
| DescribeDBInstances | 获取MongoDB实例列表                       |
| DeleteDBInstance    | 删除MongoDB实例                           |
