>注意：
>通过 mysql cli 或者程序连接创建的数据库默认授权类型为 **未授权数据库权限**。

数据库账号权限对比如下：

        <tr>
            <td >高权限账号</td>

        </tr>
        <tr>
        	<td rowspan="6">普通账号</td>
        </tr>
        <tr>
        	<td>只读</td>
            <td>SELECT, LOCK TABLES, SHOW VIEW</td>
        </tr>
